Colpitts Oscillator

INSTRUCTION

  • Part 1:Use these values and perform the experiment.
    1. Set R1=82kΩ
    2. Set R2=8.2kΩ
    3. Set R4=10kΩ
    4. Set Re=510Ω
    5. Set C1=0.1μF
    6. Set C2=0.2μF
    7. Set L1=0.5mH
    8. Set Cin=1μF
    9. Set Cout=1μF
    10. Set Ce=0.01μF
